# Dynamic Preference Depth Chart

The Dynamic Preference Depth Chart application allows you to organize and track your preferences for various everyday items, similar to a sports team's depth chart. Whether it's your favorite brand of mayo, the latest tech gadgets, or anything else, this app will help you keep track of what's currently on top and how your preferences change over time.

## Features

- **Dynamic Ranking**: Items are ranked based on your current preference, and as your preferences change, the rankings are automatically updated.
- **Versatile Use**: Use the app for anything you'd like to maintain a dynamic preference list for.
- **Responsive UI**: Built with Tailwind CSS, the app provides a responsive and modern user interface.
- **Fast Performance**: Powered by Next.js and Turbopack, the app offers a snappy user experience.
- **Persistent Data**: Utilizes an SQL database to ensure your preference history is saved and accessible.

## Usage

1. Open the app in your web browser.
2. Create a new category for your preferences.
3. Add items to the category, ranking them based on your current preference.
4. As your preferences change, simply update the rankings.

## Technologies Used

- [Tailwind CSS](https://tailwindcss.com/) for responsive and stylish UI.
- SQL database to store and manage preference data.
- [Next.js](https://nextjs.org/) for fast and dynamic web pages.
- [Turbopack](https://turbopack.dev/) for optimized module loading.
